# JOJOFriend

**JOJOFriend** is a free, open-source browser extension that automatically finds and applies the best coupon codes at checkout and helps you earn cashback at competitive rates.

[➜ Official Website](https://www.jojofriend.com/) · [➜ Privacy Policy](https://www.jojofriend.com/ext/privacy) · [➜ Supported Platforms](https://github.com/jojofriend/JOJOFriend-Shopping-Assistant/blob/main/PLATFORMS.md)

## About

### Earn Cash Back in One Click

Shop at your favorite online stores — we'll notify you whenever cashback is available. Activate your rewards with one click and keep shopping; your cashback will be on its way automatically.

### Why Choose JOJOFriend?

- **Automatic Savings** — Instantly finds and applies the best available coupon codes at checkout.
- **Browsing History Insights** — Track Amazon product history to make smarter purchasing decisions.
- **Smart Coupon Reminders** — Get notified when products have available coupons, even while searching.
- **Exclusive Deals** — Access a curated list of high-value coupons across the entire site.

Install JOJOFriend for free and start saving today — effortless discounts every time you shop.

### Transparency & Affiliate Disclosure

JOJOFriend is designed to combine convenience and savings, making it easy to claim coupons across a wide range of online shopping platforms.

This does **not** affect the discounts you receive. These partnerships help support ongoing development and allow us to keep improving the product while you enjoy real savings and a smooth shopping experience.

Your best possible shopping experience is always our top priority.

## Development

To develop new features, set up your environment with:

- **Node.js**
- **Python**
- **Visual Studio Code**
- **Visual Studio Build Tools**

### Commands

| Command | Description |
|--------|-------------|
| `npm run init` | Install all dependencies (run once before other commands). |
| `npm run update` | Update dependencies in `package.json` and install them. |
| `npm run release` | Build a minified version and output to the `dist` folder. |
| `npm run updateAndRelease` | Run `update` then `release`. |

**Init** — Install dependencies and prepare the environment:

```bash
npm run init
```

**Update** — Bump dependency versions and reinstall:

```bash
npm run update
```

**Release** — Minify source and output to `dist`:

```bash
npm run release
```
